### Quick Start

Install packages

```
yarn
```

**Client**

Run client

```
nvm use 16
yarn watch (Compile TypeScript)
```

Then, https://code.visualstudio.com/docs/editor/debugging

**Server**

Populate server `.env` vars

```
ACCESS_TOKEN_SECRET=
GITHUB_CLIENT_SECRET=
GITHUB_CLIENT_ID=
```

Run server

```
yarn watch
yarn dev
```
